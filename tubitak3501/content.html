TUBITAK 3501 Project

AI-based Interference Detection in Heterogeneous Cellular Networks

Project Number 124E765
Principal Investigator Assoc. Prof. Murtaza Cicioglu
Institution Dept. of Computer Engineering, Bursa Uludag University
Project Summary

Electromagnetic interference is one of the main factors limiting reliable communication in heterogeneous cellular networks. This project develops AI-based methods for interference detection and mitigation through feature engineering, GAN-based realistic data generation, decision-support recommendations, and an end-to-end MLOps pipeline for automated classification.
